# ESTUDO DE JAVA 7

## TIPOS DE DADOS 
  - Java suporta diferentes tipos de dados
  - O tipos de dados podem ser utilizados para criar variaveis e arrays 
  - Java é uma linguagem fortemente tipada: toda variavel e toda expressão tem um tipo e precisam ser definidos. 
     
  ##Tipos Primitivos (8 tipos)##
  
  **INTEIROS:** recebem valores inteiros
  	 byte
   	 short
   	 int
   	 long
  **PONTO FLUTUANTES:** recebem valores fracionarios
  	 float
  	 double
  **CARACTERES:** recebem carecteres letras e numeros
  		char 
  **BOOLEAN**
  		boolean
